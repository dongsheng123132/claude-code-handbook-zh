# 📕 Claude Code 红皮书 · 进阶实战与变现

> 你已经装好 Claude Code、能跑通第一个任务了（没装好先回 [📘 蓝皮书](./蓝皮书-入门.md)）。这本红皮书讲：怎么把 Claude Code 用到**能干大活、能省钱、能赚钱**。它最大的优势是**项目级编程**——整个代码库一起改，别人单文件还在折腾，你已经把活交了。

---

## 一、从"会聊天"到"会干活"：4 个心法

新手把 Claude Code 当聊天框，老手把它当**住在终端里、能 hold 住整个项目的程序员**。差别在这几点：

1. **让它在真实项目里干活，把整个库交给它**。`cd` 到你的项目根目录再开 `claude`，它能读全库所有文件、跨文件改、直接跑。Claude Code 的强项就是**大型项目**——不要把代码片段贴进对话，让它自己读整个库。
2. **一次一小步，让它先说计划**。复杂需求（整库重构、迁移框架）先说「先别动手，把方案和会改到的文件列给我看」，确认后再让它执行。改坏了好回滚——**开工前先 `git commit` 一次**，这是接项目的命根子。
3. **报错直接喂回去**。跑出来的红色报错整段复制给它：「这个报错怎么解决，改好并说明改了什么」。它会自己定位、修、重跑、再自检。
4. **善用 Skill 和子代理拆大活**。Claude Code 能把一个大任务拆给多个子代理并行干（搜索/改代码/写测试分工），还能把你常用的套路存成可复用的 Skill。复杂项目这么干，又快又不乱。具体开关敲 `claude --help` 看当前版本支持的能力。

> 💡 **省钱关键**：Claude Code 按 token 计费，让它读整个大项目很费。技巧：范围说清楚（「只动 `src/auth/` 目录」），别让它漫无目的扫全库；大改之前先让它列方案、确认了再放它跑，避免来回烧 token。

## 二、5 个实战场景（照着改就能用）

### 场景 1 · 大型项目改造：一次 hold 住整个库
Claude Code 最能打的地方。把整个老项目交给它，跨几十个文件协同改：

> 这是一个现成的网站项目，帮我加一个完整的「用户登录 + 注册」功能：前端页面、后端接口、数据库表都要。先把方案和会改到的文件全列给我，我确认后再动手，改完跑一遍测试。

它会读懂全库结构、按你的代码风格跨文件改、改完自检。**接私活做整个项目**的主力玩法。

### 场景 2 · 整库重构 / 框架迁移
> 把这个项目从 Vue2 整体迁移到 Vue3（含 Composition API 改写），逐个组件改，每改完一批就告诉我进度，最后确认能跑起来。

老代码迁移、技术栈升级、批量重构——这类活人工干又慢又容易漏，Claude Code 跨文件一致地改，单价高、需求还旺。

### 场景 3 · 给老项目加功能 / 修 bug
> 线上这个错：[粘贴整段报错]。帮我定位原因，只做必要修改，改完说明改了哪些文件、怎么验证。

读懂现有结构、按你的代码风格改、改完能跑测试。**接私活改代码**最高频的需求，一单几十到几百块。

### 场景 4 · 写测试 / 做代码审查
不想自己写一堆测试、或想给项目找隐患？

> 给 `src/` 下所有核心模块写单元测试，覆盖正常和异常情况，并运行；顺便做一次代码审查，列出潜在 bug 和安全风险。

补测试、查漏洞、做 review——给小团队做"质量外包"，按项目收费，门槛不高、复购强。

### 场景 5 · 接私活做完整项目
> 帮我做一个「小商家收银 + 库存管理」的小系统：能开单、记库存、出日报表，本地能跑，做完告诉我怎么启动和打包。

从 idea 到能交付的完整项目，Claude Code 一个人就能跟下来。猪八戒 / 闲鱼 / 朋友圈上，这类「做个小系统」的单子 500–5000 元一单，你接得住。

## 三、提示词模板库（复制改需求即用）

| 目的 | 模板 |
|---|---|
| 让它先规划 | `先别动手，针对「___」给我 2 个方案 + 各自优缺点 + 会改到的文件，我选一个你再做。` |
| 安全改代码 | `只改「___目录/文件」，实现「___」，不要动其它部分，改完跑一下确认没报错。` |
| 整库重构 | `把这个项目从「___」迁到「___」，逐文件改，每完成一批汇报进度，最后确认能跑。` |
| 修 bug | `运行报这个错：[粘贴报错]。帮我定位原因并只做必要修改，说明改了什么、怎么验证。` |
| 读懂项目 | `把整个项目读一遍，用大白话讲它是干嘛的、入口在哪、核心逻辑在哪几个文件、有哪些坑。` |
| 写测试 | `给「___」写单元测试，覆盖正常和异常情况，并运行；失败的帮我一起修好。` |

> 想要按行业分类的完整模板包（电商、外包接单、企业内部工具），扫码找我领。

## 四、变现路径：Claude Code 怎么帮你赚钱

| 路径 | 怎么做 | 门槛 |
|---|---|---|
| **接私活做项目** | 闲鱼、猪八戒、朋友圈接「做个小系统 / 小程序 / 网站」单，Claude Code 帮你跟全程 | 中，会描述需求 + 懂点基础更稳 |
| **改代码 / 修 bug** | 给小商家改网站、修小程序 bug、加功能，按单收费 | 中 |
| **整库重构 / 迁移** | 帮公司迁框架、升技术栈、批量重构老代码，单价高 | 中高 |
| **代码审查 / 测试外包** | 给小团队做质量把关、补测试、查漏洞，按项目收 | 中 |
| **知识付费 / 教学** | 教别人用 Claude Code（就像这本手册），引流到社群变现 | 低，会用 + 会讲 |

**核心逻辑**：Claude Code 把"做一个完整项目"的门槛从"会编程一两年"降到"会描述需求 + 会验收"。你赚的是**信息差 + 执行力**的钱——大多数人还不知道一个人就能交付整个项目。
