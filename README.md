## 这是什么

[Claude Code](https://github.com/anthropics/claude-code) 是 Anthropic 官方的**命令行 AI 编程工具**（npm 包 `@anthropic-ai/claude-code`）——在终端里用大白话指挥它读整个项目、写代码、改 bug、跑命令、自己改自己跑。它是目前**最强的项目级 AI 编程工具**之一，但国内用有三个坑：

1. **装不上**——npm 走默认源容易卡死/超时，客户机上的 clash/v2ray 代理还常把 npm 拦死，结果命令半天装不下来。
2. **要登录 Anthropic**——官方默认要登 Anthropic 账号，国内注册、网络、支付样样是门槛。
3. **不会配模型**——Anthropic 官方额度国内不好充，不挂梯子根本连不上。

这本手册把这三个坑全部填平，并给你一条**最省事的路**：用 [U-King](https://www.u-king.org/?from=claude-code) 一键装好、接好国内可用的 **虾盘云驱动**（¥1 ≈ 50 万 token，Claude Code 走 **DeepSeek-V4 Pro 满血**，不用注册 Anthropic、不用翻墙），双击即用。
